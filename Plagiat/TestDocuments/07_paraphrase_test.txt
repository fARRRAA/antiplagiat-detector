Тест перефразирования для AI системы

Оригинальный текст для перефразирования:

Глобальное потепление представляет собой одну из наиболее серьезных экологических проблем современности. Повышение средней температуры на Земле приводит к таянию ледников, повышению уровня морей и изменению климатических условий во всем мире.

Основными причинами глобального потепления являются:
1. Выбросы парниковых газов от промышленности
2. Сжигание ископаемого топлива
3. Вырубка лесов
4. Интенсивное сельское хозяйство

Последствия изменения климата включают:
• Экстремальные погодные явления
• Угрозу биоразнообразию
• Проблемы продовольственной безопасности
• Миграцию населения

Для борьбы с глобальным потеплением необходимо:
- Переход на возобновляемые источники энергии
- Повышение энергоэффективности
- Развитие экологически чистых технологий
- Международное сотрудничество

Этот текст можно использовать для тестирования различных уровней перефразирования:
• Легкое перефразирование (замена синонимов)
• Среднее перефразирование (изменение структуры предложений)
• Глубокое перефразирование (полная переформулировка с сохранением смысла)
